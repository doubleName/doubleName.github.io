# img导入拼接地址遇到的问题

模板字符串不好使了咋办==

可以把它写成一个方法，绑定，使用方法来拼接字符串，然后输出字符串。尝试之后可行。

 <img :src="toSrc(item.img)">


 methods: {
      toSrc: function(string) {
        return `http://localhost:8080/img/${string}`
      }
}
